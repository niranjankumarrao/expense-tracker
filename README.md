                                                                          Bellcorp Personal Expense Tracker (MERN Stack)
 
Project Overview

This is a Personal Expense Tracker web application built using the MERN stack (MongoDB, Express, React, Node.js).
The application allows users to securely register, login, and manage their daily expenses.
Users can add, edit, delete, and view transactions along with a dashboard summary.

Tech Stack

Frontend: React.js, React Router, Axios
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Token)

Features

Authentication
User Registration
User Login
Protected Routes (only logged-in users can access Dashboard)
Transaction Management

Users can:

Add new transaction
Edit existing transaction
Delete transaction
View transaction details
Each transaction contains:
Title
Amount
Category
Date
Notes (optional)

Dashboard

Total expenses summary
Category-wise expense breakdown
Recent transactions preview

Transaction Explorer

Browse large transaction history
Search transactions by text
Handles empty results
